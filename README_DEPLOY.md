## ❓ 常见问题

### Q: 一定要买服务器吗？
**A:** 不一定！可以使用免费服务（Railway、Render）或本地电脑测试。

### Q: 最便宜的方案是什么？
**A:** 
- 免费：Railway 免费额度
- 付费：阿里云轻量服务器 ¥24/月

### Q: 国内访问快吗？
**A:** 
- 国内服务器（阿里云、腾讯云）：国内访问快
- 国外服务器：可能较慢，建议使用 CDN

### Q: 需要备案吗？
**A:** 
- 使用国内服务器 + 域名：需要备案
- 使用 IP 地址访问：不需要备案
- 使用国外服务器：不需要备案
