## 文件命名规则

- **音频文件**：武器名称 + `.mp3` 扩展名
- **视频文件**：武器名称 + `.mp4` 扩展名
- 文件名必须与 `EquipmentMedia.ts` 中配置的名称完全一致

## 使用方法

1. **添加音频文件**：
   - 将音频文件（MP3格式）放入 `public/audio/` 目录
   - 文件名必须与武器名称完全匹配

2. **添加视频文件**：
   - 将视频文件（MP4格式）放入 `public/videos/` 目录
   - 文件名必须与武器名称完全匹配

3. **功能说明**：
   - 点击展品后，会显示信息面板
   - 点击"语音讲解"按钮或按键盘 **1** 键播放语音
   - 点击"视频介绍"按钮或按键盘 **2** 键播放视频
   - 如果音频文件不存在，系统会自动使用浏览器TTS（文字转语音）功能

## 注意事项

- 音频文件建议使用 MP3 格式，文件大小控制在 5MB 以内
- 视频文件建议使用 MP4 格式（H.264编码），文件大小建议控制在 50MB 以内
- 确保文件名与代码中的武器名称完全一致（包括中文字符）
- 如果文件不存在，系统会尝试使用浏览器内置的TTS功能作为备选方案
