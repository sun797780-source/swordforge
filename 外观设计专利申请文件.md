## 二、简要说明（正式版，可直接提交）

```
【简要说明】

1. 产品名称：带有兵工文化教育展示界面的显示屏幕面板。

2. 用途：本外观设计产品用于兵工文化教育体验平台的图形用户
界面显示。用户可通过该图形用户界面进行兵工历史信息的交互
式浏览、装备知识的分类查询与学习，以及基于人工智能的装备
性能仿真分析等人机交互操作。

3. 设计要点：设计要点在于显示屏幕面板上显示的图形用户界面
的图案与色彩设计。具体包括：
（1）主视图中，左侧纵向排列的历史阶段时间轴信息面板与
装备类型统计区域的双层信息架构设计，以及右侧三维虚拟展厅
场景中装备展品标签的布局设计；
（2）使用状态变化图中，顶部功能输入区域、中部三维轨迹
可视化区域与右侧参数评估面板的三栏组合式布局设计，以及
底部预设方案选择卡片的排列设计；
（3）整体深色背景与金色强调色的配色方案。

4. 指定一幅最能表明设计要点的图片或照片：主视图（图1）。

5. 省略视图及原因：本产品为平板状显示屏幕面板，其左视图与
右视图对称且无设计要点，俯视图与仰视图为窄边平面且无设计
要点，故省略左视图、俯视图和仰视图。

6. 请求保护色彩：请求保护主视图和使用状态变化图中所示的
图形用户界面的色彩。
```

## 五、提交流程（手把手教程）

### 第1步：注册电子申请账号（第1天）

```
1. 访问 https://cponline.cnipa.gov.cn
2. 点击页面右上角"注册"
3. 选择"个人注册"
4. 填写信息：
   - 真实姓名
   - 身份证号
   - 手机号
   - 常用邮箱
5. 设置登录密码
6. 完成注册后，下载并安装 CPC客户端
   下载地址：https://cponline.cnipa.gov.cn/toolDownload/toolDownload
```

### 第2步：提交费减备案（第1-3天）

```
1. 访问 http://cpservice.cnipa.gov.cn
2. 使用刚注册的账号登录
3. 在左侧菜单找到"专利费减备案"
4. 填写信息：
   - 身份类型：个人
   - 年收入：如实填写（学生填0即可）
5. 上传证明材料：
   - 学生身份证明（学生证拍照/扫描件）
   - 或收入证明文件
6. 提交后等待审核（1-3个工作日）
7. 审核通过后，后续申请自动减缓85%费用

⚠️ 费减备案必须在提交专利申请之前完成！
```

### 第3步：准备申请文件（第3-5天）

```
需要准备的文件清单：
  □ 外观设计专利请求书（CPC客户端中在线填写）
  □ 图1 — 主视图.jpg
  □ 图2 — 使用状态变化图.jpg
  □ 图3 — 后视图.jpg
  □ 图4 — 右侧视图.jpg
  □ 简要说明.doc（复制本文档第二部分的内容）
```

### 第4步：通过CPC客户端提交（第5-7天）

```
1. 打开CPC客户端，登录账号
2. 点击"案件管理" → "新建" → "外观设计专利申请"
3. 填写请求书：
   - 产品名称：带有兵工文化教育展示界面的显示屏幕面板
   - 类别：14-04
   - 申请人信息：[你的信息]
   - 设计人信息：[你和团队成员]
4. 上传视图图片（4张）
5. 上传简要说明文件
6. 点击"文件检查"确认无误
7. 点击"签名" → "提交"
8. 系统会返回一个申请号（格式：2026XXXXXXXX.X）
9. 保存好这个申请号！
```

### 第5步：缴纳申请费（提交后15天内！）

```
⚠️ 这是最重要的截止日期，逾期视为撤回申请！

缴费方式（推荐网上缴费）：
1. 访问 http://fee.cnipa.gov.cn
2. 登录账号
3. 输入申请号
4. 选择"申请费"
5. 确认金额：75元（费减后）
6. 在线支付（支持银行卡、支付宝等）
7. 保存缴费凭证
```

### 第6步：等待审查（3-6个月）

```
期间需要做的：
  - 每周登录CPC客户端查看是否有通知
  - 如果收到"补正通知"→ 按要求修改后2个月内重新提交
  - 如果收到"授权通知"→ 进入第7步

查看进度：
  登录 https://cponline.cnipa.gov.cn → 我的案件 → 查看状态
```

### 第7步：缴纳登记费 + 拿证（收到授权通知后）

```
收到《授权通知书》后 2个月内：

缴费项目：
  ├── 登记费：205元（不减缓）
  ├── 印花税：5元
  ├── 第1年年费：600元 × 15% = 90元（费减后）
  └── 合计：约 300元

缴费后约 1-2个月：
  → 收到《外观设计专利证书》📜
  → 专利号格式：ZL 2026 3 XXXXXXX.X
```


## 九、如果被要求补正怎么办

常见的补正要求及应对方法：

| 补正要求 | 怎么改 |
|---------|--------|
| 图片不清晰 | 重新截取更高分辨率的截图 |
| 视图之间比例不一致 | 统一所有视图中显示器轮廓的大小比例 |
| 产品名称不规范 | 改成"带有XX界面的显示屏幕面板"格式 |
| 简要说明与视图不一致 | 核对简要说明中的描述是否与图片内容对应 |
| 缺少视图 | 补充后视图或侧视图 |
